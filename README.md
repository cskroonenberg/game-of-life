# game-of-life

game-of-life is a C program which initializes and runs a simulation of Conway's Game of Life

## Initialization

```bash
# Create build directory
mkdir build
cd build

# Build project
cmake .. && make

# Execute program with init files from samples directory
./gol ../samples/<SAMPLE>
```
